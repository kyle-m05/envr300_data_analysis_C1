**Temperature trends and projections of residential and commercial natural gas 
consumption in North Carolina**


This is a data analysis project on the trends and forecasted trends of natural 
gas consumption in NC. With data from 1989 to 2023 and forecasted temperature 
data from 5 climate models. 

Packages (only need to install once):
#install for tidyverse
install.packages("tidyverse")

#install for the here package (easier for file locating)
install.packages("here")

Project Directory:
- 'data/' - LOAD THIS FIRST contains all the datasets and libraries required!
